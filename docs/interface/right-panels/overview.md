Рабочая область содержит в себе всё необходимое для работы с проектом:

-   View panel
-   Панель добавления
-   Pages and layers
-   Code Editor
-   Page
-   Выбор элементов
-   Theme and Project Settings
-   Element props panel (when an element is selected)
-   Контекстное меню
