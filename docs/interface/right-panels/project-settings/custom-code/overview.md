Вкладка Custom Code, отвечает за добавление тегов внутрь head и в конец body.

Теги которые можно добавлять внутрь head:

-   &lt;Script&gt;
-   &lt;Link&gt;
-   &lt;Meta&gt;
-   &lt;Style&gt;

Теги которые можно добавлять в конец body:

-   &lt;Script&gt;
-   &lt;Link&gt;
-   &lt;Style&gt;
